# 🎬 Movie Search App

A simple React-based movie search application that uses the OMDb API to fetch and display movie data.

---

## 🚀 Features

- 🔍 Search movies by name
- 🎞️ Displays movie title, poster, and year
- ⚡ Fast and responsive UI
- 🌐 API integration with OMDb

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript
- CSS
- OMDb API
